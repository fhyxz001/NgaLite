# NgaLite 项目设计书（初版）

## 1. 项目概述

NGA 论坛轻量客户端，基于 Kotlin + Jetpack Compose 的原生 Android 应用。
初版仅实现「帖子列表 → 帖子详情」两级浏览，登录采用复制 Cookie 的方式。

## 2. 技术选型

| 项 | 选择 |
|----|------|
| 语言 | Kotlin 2.2.10 |
| UI | Jetpack Compose + Material3 |
| 构建 | Gradle 9.4.1 + AGP 9.2.1 (Kotlin DSL) |
| 网络 | OkHttp 4 |
| 解析 | Jsoup（支持 GBK 编码） |
| 导航 | navigation-compose |
| 最低 SDK | 26（Android 8.0，自适应图标） |
| 目标 SDK | 34 |

## 3. 功能范围

仅以下三项，不做扩展：

1. **Cookie 登录**：设置入口粘贴 Cookie 字符串，持久化到 SharedPreferences，请求时携带。
2. **帖子列表**：拉取指定版块帖子，展示标题/回复数/作者/最后回复时间，点击进入详情。
3. **帖子详情**：展示楼层、作者、时间、正文（HTML 转纯文本）。

## 4. 页面结构

```
MainActivity (单 Activity)
├── ListScreen   帖子列表  ——  顶栏含「登录」入口
└── DetailScreen 帖子详情  ——  接收 tid 参数
```

导航：ListScreen →(点击条目, 传 tid)→ DetailScreen。

## 5. 数据来源

NGA 论坛（GBK 编码 HTML）：

- 列表：`https://bbs.nga.cn/thread.php?fid=-7955747`
  - 解析 `<tr class="topicrow">`
  - tid：`<a href="read.php?tid=X">`
  - 标题：`<a class="topic">`
  - 回复数：`<a class="replies">`
  - 最后回复：`<a class="replydate">` / `<span class="replyer">`
- 详情：`https://bbs.nga.cn/read.php?tid={tid}`
  - 解析 `<tr class="postrow">`
  - 作者：`#postauthorN`
  - 时间：`#postdateN`
  - 正文：`#postcontentN`（HTML→文本）

## 7. 登录方案

- 用户在浏览器登录 NGA 后，复制请求 Cookie（含 `ngaPassportUid` 等）。
- 应用内「登录」对话框粘贴保存，存入 `SharedPreferences`。
- 所有 NGA 请求附带 `Cookie` 头与桌面 `User-Agent`。
- 不实现账号密码登录、验证码、Token 刷新等。

## 8. 不做的事（初版边界）

不做：发帖/回帖、收藏、消息、搜索、分页、图片加载富文本、夜间模式、缓存、下拉刷新、错误重试。
